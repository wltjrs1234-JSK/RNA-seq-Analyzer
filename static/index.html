Mutant_TPM AVG Mutant 평균 TPM (또는 FPKM, Count) 233690.32

Gene Ontology Enrichment (유전자 온톨로지 농축 분석)

선택한 유전자군에 대해 SGD Yeast GO Slim 데이터셋을 기반으로 Hypergeometric test(초기하 분포 통계 검정)를 수행하여 통계적으로 유의미한 기능 범주를 도출합니다.

GO ID 분류 (Aspect) 기능 범주 (Term Name) 매칭 (k) 전체 (M) p-value FDR (BH) 매칭 유전자 목록
분석 실행 버튼을 누르시면 검정이 시작됩니다.

Protein Domain (Pfam/InterPro) 및 기능 도메인 농축 결과

선택된 DEG 유전자군이 보유한 단백질 패밀리 도메인 분포를 분석하여 특정 분자 도메인이 통계적으로 유의미하게 농축되어 있는지 분석합니다.

도메인 ID (Pfam) 도메인 설명 (Domain Name) 매칭 (k) p-value FDR (BH)
분석 실행 버튼을 누르시면 도메인 결과가 함께 연산됩니다.

PCA (주성분 분석) - 샘플 변동성

전체 유전자 발현 데이터를 주성분 분석(PCA)으로 차원 축소하여 WT 3개 반복구와 Mutant 3개 반복구 간의 분리도와 발현 변동성(Variance)을 시각화합니다.
